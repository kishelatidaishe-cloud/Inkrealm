# InkRealm - Interactive Book Platform

© 2026 Tawananyasha Nzombe. All Rights Reserved.

InkRealm is a comprehensive interactive website for discovering, reading, and sharing novels and books. Explore our extensive library, discover original works, and connect with fellow book lovers.

## Features

- 📚 **Extensive Library** - Thousands of novels and books organized by genre, author, and popularity
- ✍️ **Original Novels** - Exclusive works published by Tawananyasha Nzombe
- 👤 **User Profiles** - Create personalized reading profiles and track your library
- ⭐ **Ratings & Reviews** - Share your thoughts and discover community recommendations
- 📖 **Reading Lists** - Curate your own reading collections and wishlists
- 🔍 **Advanced Search** - Filter by genre, author, release date, and ratings
- 🎨 **Beautiful UI** - Responsive, modern design for all devices
- 📱 **Mobile Friendly** - Access InkRealm on any device

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
